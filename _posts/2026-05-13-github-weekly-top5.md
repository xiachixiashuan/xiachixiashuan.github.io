# GitHub 周榜 Top5｜2026-05-13

本周 GitHub 涌现了一批令人眼前一亮的新项目——从腾讯 AI 实验室的 SIGGRAPH 论文级 3D 生成，到用 Rust 写的 Discord 终端客户端，再到专为中文网文作者打造的 AI 写作引擎。以下是过去 7 天里星数增长最快的 5 个值得关注的开源项目。

## 一、cclank/cell-architecture-studio

⭐ 434 | TypeScript | 作者：cclank

一个基于 React + Three.js 构建的交互式 3D 细胞架构展览馆。项目将生物细胞结构以三维可视化形式呈现，用户可在浏览器中实时旋转、缩放、探索各类细胞构件。对生物信息学、科学可视化感兴趣的开发者来说是难得的参考实现，代码结构也相当清晰。

[项目地址](https://github.com/cclank/cell-architecture-studio)

## 二、chojs23/concord

⭐ 300 | Rust | 作者：chojs23

用 Rust 编写的功能完备 TUI（终端界面）Discord 客户端。支持频道浏览、消息收发、通知等核心功能，完全在命令行环境运行，无需 Electron。对于热爱终端工作流、希望告别重量级 Discord 客户端的开发者来说，这是一个令人期待的选择。

[项目地址](https://github.com/chojs23/concord)

## 三、kunchenguid/autopreso

⭐ 291 | JavaScript | 作者：kunchenguid

"让白板自己白板自己"——autopreso 是一个将实时语音转化为演示幻灯片的 AI 工具。用户边说话，幻灯片内容边自动生成，彻底颠覆传统 PPT 制作流程。适合演讲者、教师和产品经理在会议或直播中即兴展示想法。

[项目地址](https://github.com/kunchenguid/autopreso)

## 四、TencentARC/Pixal3D

⭐ 262 | Python | 作者：TencentARC

腾讯 ARC 实验室发布的 SIGGRAPH 2026 论文实现——Pixal3D：从单张或多张图像进行像素对齐的 3D 生成。相比传统 NeRF 或 3DGS 方案，Pixal3D 在像素级精度和生成速度上均有显著提升，是图像到 3D 领域的前沿研究成果，代码已完整开源。

[项目地址](https://github.com/TencentARC/Pixal3D)

## 五、zy-zmc/tianming-novel-ai-writer

⭐ 130 | C# | 作者：zy-zmc

专为中文网文作者设计的 AI 辅助写作系统「天命」。采用 12 维事实快照 + 9 类变更声明 + 6 道生成门禁的架构，彻底解决 AI 写长篇小说时的前后矛盾问题。不依赖模型记忆，而是通过每章状态回写保持世界观一致性，号称写到 3000 章依然连贯。

[项目地址](https://github.com/zy-zmc/tianming-novel-ai-writer)
