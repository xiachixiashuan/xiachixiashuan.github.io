# GitHub 周榜 Top5｜2026-04-30

本周 GitHub 涌现了一批令人眼前一亮的开源项目，从 AI 辅助设计工具到 PS5 破解引导器，技术圈的创造力从未止步。以下是过去 7 天新诞生、星数最高的 5 个项目。

## 一、nexu-io/open-design

⭐ 4340 ｜ TypeScript ｜ nexu-io

open-design 是 Anthropic Claude Design 的本地优先开源替代品，内置 19 个设计技能和 71 套品牌级设计系统，支持沙盒预览和 HTML / PDF / PPTX 多格式导出。它兼容 Claude Code、Codex、Cursor、Gemini CLI、Qwen 等主流 AI 编码工具，无需联网即可在本地全量运行，是设计工程师打造 AI 驱动工作流的理想底座。

[项目地址](https://github.com/nexu-io/open-design)

## 二、cursor/cookbook

⭐ 1302 ｜ TypeScript ｜ Cursor 官方

cursor/cookbook 是 Cursor AI 官方发布的实战案例库，收录了使用 Cursor 与 AI 模型完成各类软件工程任务的可复现示例。对于想系统学习如何用 AI IDE 提升开发效率的工程师来说，这本"食谱"提供了最权威的第一手参考，覆盖前端、后端、全栈多种开发场景。

[项目地址](https://github.com/cursor/cookbook)

## 三、ps5-linux/ps5-linux-loader

⭐ 516 ｜ C ｜ ps5-linux 组织

ps5-linux-loader 利用 PS5 上的 HV（Hypervisor）漏洞实现了一个自定义引导程序，可以在索尼 PlayStation 5 上加载并运行 Linux 系统。这是游戏主机 Linux 移植领域的重要里程碑，为硬件爱好者和安全研究者打开了一扇新的大门，展示了现代游戏主机底层架构的研究潜力。

[项目地址](https://github.com/ps5-linux/ps5-linux-loader)

## 四、NyxTides/ppt-image-first

⭐ 421 ｜ Python ｜ NyxTides

ppt-image-first 是一个专为 Claude Code、Codex 和 Opencode CLI 设计的 PPT 生成技能，采用"图像优先"理念，直接输出以视觉内容为主体的幻灯片。告别枯燥的纯文字演示，只需一条指令，AI 便能自动完成配图排版，生成图文并茂的高质量演示文稿。

[项目地址](https://github.com/NyxTides/ppt-image-first)

## 五、GENEXIS-AI/chromex

⭐ 406 ｜ TypeScript ｜ GENEXIS-AI

chromex 是一个由 Codex 驱动的 Chrome 侧边栏 AI 助手，能够感知当前页面内容、管理多标签上下文、处理语音输入和图片工作流。它将 AI 编程助手的能力直接嵌入浏览器侧边栏，让日常网页阅读、信息提取和多任务操作变得更加智能高效。

[项目地址](https://github.com/GENEXIS-AI/chromex)
