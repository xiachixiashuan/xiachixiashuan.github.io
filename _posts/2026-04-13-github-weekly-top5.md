# GitHub 周榜 Top5｜2026-04-13

本周 GitHub 新项目热潮不减，从 Hermes AI 代理 Web 监控界面、Linux 网络防火墙、ML 刷题平台，到 AI 技能自进化框架和让大模型"说人话"的系统提示词，五个方向各有惊喜。以下是本周诞生、已获大量关注的五个精选开源项目。

## 一、joeynyc/hermes-hudui

⭐ 701 | 语言：Python | 作者：joeynyc

Hermes HUD UI 是专为 Hermes AI 代理设计的 Web 端"意识监控仪表盘"，实时可视化这个具备持久记忆的 AI 代理的运行状态。你可以通过它追踪代理的记忆写入、任务执行链路与上下文状态，**将原本黑箱的代理行为变为可观测、可调试的透明流程**。对于正在构建或研究 AI 代理系统的开发者，这是一个不可多得的调试与展示工具。

[项目地址](https://github.com/joeynyc/hermes-hudui)

## 二、obdev/littlesnitch-linux

⭐ 601 | 语言：Rust | 作者：obdev

macOS 经典网络防火墙工具 Little Snitch 正式推出 Linux 版本的开源组件。基于 Rust 构建，提供**进程级出入站流量监控与拦截**能力。长期以来，Linux 桌面在精细化网络管控方面远落后于 macOS；这次 obdev 官方入场，意味着 Little Snitch 品质的网络可视化能力首次登陆 Linux，值得安全研究人员和隐私意识强的开发者重点关注。

[项目地址](https://github.com/obdev/littlesnitch-linux)

## 三、whwangovo/pyre-code

⭐ 579 | 语言：Python | 作者：whwangovo

pyre-code 是一个自托管的机器学习编程练习平台，内置 68 道精心设计的题目，覆盖从 ReLU 激活函数到流匹配（Flow Matching）的完整 ML 知识链：注意力机制、Transformer 训练、RLHF、扩散模型等一网打尽。所有题目支持**在浏览器内即时反馈**，无需本地环境配置，非常适合系统性补齐 ML 底层实现能力的工程师和研究人员。

[项目地址](https://github.com/whwangovo/pyre-code)

## 四、AMAP-ML/SkillClaw

⭐ 437 | 语言：Python | 作者：AMAP-ML

SkillClaw 是一个让 AI 技能（Skill）**通过代理进化器集体自我迭代**的框架。它将技能开发流程自动化：代理自主运行技能、评估输出、发现缺陷并提出改进版本，形成闭环的技能进化回路。这一思路将 AI 对齐和能力提升从人工迭代推向了部分自动化，对 AI 基础设施研究者而言是一个有价值的实验平台。

[项目地址](https://github.com/AMAP-ML/SkillClaw)

## 五、hexiecs/talk-normal

⭐ 317 | 语言：Shell | 作者：hexiecs

一个精简的系统提示词集合，目标只有一个：**让任何 LLM 像正常人一样说话**，彻底去除"AI 腔"（AI slop）。不再有"当然！""很高兴您能问这个问题！"这类空洞开场，不再有冗余的免责声明与格式堆砌。对于日常大量使用 AI 助手的用户，这份提示词值得直接放进你的系统配置文件，立竿见影地提升对话质量。

[项目地址](https://github.com/hexiecs/talk-normal)
