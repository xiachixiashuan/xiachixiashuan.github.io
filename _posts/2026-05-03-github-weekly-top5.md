# GitHub 周榜 Top5｜2026-05-03

本周 GitHub 上又涌现出一批有趣的新项目，从 macOS 实用工具到 AI 编程辅助，从数据库客户端到中国开发者专属神器，五个项目各有亮点。以下是本周星数最高的 5 个新仓库速览。

## 一、darrylmorley/whatcable

⭐ 1.3k｜Swift｜作者：Darryl Morley

你有没有遇到过插上一根 USB-C 线却不知道它支持什么功能的困扰？WhatCable 是一款 macOS 菜单栏小工具，插上 USB-C 线后，它会用通俗英语告诉你这根线到底能做什么：充电功率、视频输出、数据传输速度……一目了然。它基于 IOKit 读取硬件信息，支持苹果芯片，用 SwiftUI 开发，界面清爽原生。对于经常被各种 USB-C 线搞晕的 Mac 用户来说，这是一个解决痛点的小而美工具。

[项目地址](https://github.com/darrylmorley/whatcable)

## 二、b-nnett/codex-plusplus

⭐ 836｜TypeScript｜作者：b-nnett

Codex++ 是专为 OpenAI Codex 桌面应用设计的 tweak 扩展系统，允许用户在不修改官方客户端的前提下注入自定义功能和优化配置。它类似于 iOS 越狱工具圈里的 tweak 机制，让高级用户可以突破默认限制，按自己的需求定制 Codex 的行为。项目使用 TypeScript 构建，结构清晰，对 Codex 重度用户来说扩展性极强。

[项目地址](https://github.com/b-nnett/codex-plusplus)

## 三、t8y2/dbx

⭐ 712｜Vue + Rust｜作者：t8y2

dbx 是一款轻量级、跨平台的数据库客户端，底层基于 Tauri（Rust），前端使用 Vue 构建。它一口气支持 MySQL、PostgreSQL、SQLite、Redis、MongoDB、DuckDB、ClickHouse、SQL Server 等主流数据库，提供统一的图形化操作界面。与臃肿的企业级工具相比，dbx 启动快、资源占用低，非常适合独立开发者和全栈工程师日常使用。

[项目地址](https://github.com/t8y2/dbx)

## 四、mattpocock/dictionary-of-ai-coding

⭐ 677｜TypeScript｜作者：Matt Pocock

AI 编程领域行话满天飞：vibe coding、agentic loop、context window……这个「AI 编程词典」项目由 TypeScript 教程知名博主 Matt Pocock 发起，用大白话解释 AI 编程中常见的专业术语。每个词条都有清晰的定义和使用场景，帮助开发者快速跟上 AI 编程范式转变的节奏。对于刚进入 AI 辅助开发领域的工程师，这是一本必备的入门参考手册。

[项目地址](https://github.com/mattpocock/dictionary-of-ai-coding)

## 五、Fokkyp/SoftwareCopyright-Skill

⭐ 636｜Python｜作者：Fokkyp

中国软件著作权申请流程繁琐，材料格式要求严格，动辄要花几百元让中介代办。这个项目提供了一个全开源的软著申请材料生成器：它读取你的本地项目代码，自动生成符合国家版权局格式要求的 .docx 文档，包括代码清单、操作说明书等全套材料。完全免费，无需任何付费服务，专为中国独立开发者量身定制。

[项目地址](https://github.com/Fokkyp/SoftwareCopyright-Skill)
