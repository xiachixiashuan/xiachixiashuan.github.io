# GitHub 周榜 Top5｜2026-04-20

本周 GitHub 榜单新秀辈出：3D 场景基础模型、LLM 自愈浏览器框架、安全漏洞研究库、Claude 架构开源复现以及量化投资 AI 技能包，五个方向各有亮点，均诞生于过去 7 天。

## 一、Robbyant/lingbot-map

⭐ 2.6k｜语言：Python｜作者：Robbyant

lingbot-map 是一个前馈式 3D 基础模型，专为从流式传感器数据中实时重建三维场景而设计。与传统需要迭代优化的方法不同，它直接以前馈推理一步完成重建，速度显著更快。适合自动驾驶、机器人感知等对实时性要求极高的三维理解场景。

[项目地址](https://github.com/Robbyant/lingbot-map)

## 二、browser-use/browser-harness

⭐ 2.2k｜语言：Python｜作者：browser-use

browser-harness 是一个"自愈式"浏览器操控框架，让大语言模型能够执行任意复杂的 Web 任务。当目标页面结构发生变化时，框架会自动修复失效的选择器，无需人工介入。这是 browser-use 团队继同名项目后的新力作，专为生产级 AI 自动化场景设计，稳定性大幅提升。

[项目地址](https://github.com/browser-use/browser-harness)

## 三、Nightmare-Eclipse/RedSun

⭐ 1.6k｜语言：C++｜作者：Nightmare-Eclipse

RedSun 是一个专注于漏洞研究的仓库，收录了安全研究员披露的漏洞利用代码与详细技术分析。对安全研究者和 CTF 竞赛选手而言，这是一份高质量的攻防参考资源；同时也可用于防御侧的漏洞机制理解与修复研究。

[项目地址](https://github.com/Nightmare-Eclipse/RedSun)

## 四、kyegomez/OpenMythos

⭐ 1.4k｜语言：Python｜作者：kyegomez

OpenMythos 基于公开研究文献，从第一性原理对 Claude Mythos 架构进行理论性开源复现。项目涵盖注意力机制变体、循环 Transformer 等前沿结构探索，致力于透明可解释的架构设计，是研究下一代大模型架构的极佳实验平台。

[项目地址](https://github.com/kyegomez/OpenMythos)

## 五、wbh604/UZI-Skill

⭐ 0.8k｜语言：Python｜作者：wbh604

UZI-Skill 是一个面向 A 股、港股、美股的量化投资 AI 技能包，集成了 51 位投资大佬的看盘逻辑、22 维行情数据维度、180 条量化规则与 17 种机构级别分析方法。适合希望用 AI 辅助投资决策的开发者和量化爱好者，快速搭建个人选股分析系统。

[项目地址](https://github.com/wbh604/UZI-Skill)
