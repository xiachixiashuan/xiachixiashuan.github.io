# GitHub 周榜 Top5｜2026-04-11

过去 7 天，GitHub 上诞生了一批令人眼前一亮的新项目。本期周榜精选 5 个新星仓库，涵盖 AI Agent、开发者工具、macOS 生产力应用等多个热门赛道，一起来看看这周最值得关注的开源力量。

---

## 一、xixu-me/awesome-persona-distill-skills

⭐ 2.8k｜语言：JavaScript｜作者：xixu-me

"同事.skill、女娲.skill、前任.skill……" 这个 Awesome List 汇聚了所有以**真实人物、人际关系与方法论视角**为核心的 Agent Skill，是目前最全的人格蒸馏 Skills 导航站。无论你想蒸馏某位创业者的思维方式，还是还原某段关系的沟通模式，这里都有对应的入口。对 Claude Skills 生态感兴趣的开发者强烈推荐收藏。

[项目地址](https://github.com/xixu-me/awesome-persona-distill-skills)

---

## 二、garrytan/gbrain

⭐ 2.5k｜语言：TypeScript｜作者：garrytan

gbrain 是一套为 OpenClaw / Hermes Agent 框架设计的**固执己见的 Agent 大脑**，内置了一套经过打磨的推理规划与任务调度逻辑。它的设计哲学是：与其让模型自由发挥，不如给它一个高质量的"操作系统"。TypeScript 实现，适合希望在 Hermes 生态上快速搭建稳定 Agent 工作流的开发者。

[项目地址](https://github.com/garrytan/gbrain)

---

## 三、hotcoffeeshake/tong-jincheng-skill

⭐ 1.5k｜语言：N/A（Skill 配置文件）｜作者：hotcoffeeshake

将"深情祖师爷"童锦程的人际关系思维框架蒸馏为一个可调用的 Agent Skill。用这个 Skill，你可以从童锦程视角分析感情、商业与人性，体验一种极具辨识度的**情感分析与决策风格**。人格蒸馏赛道的又一力作，验证了 Skills 生态在非技术内容创作方向的巨大潜力。

[项目地址](https://github.com/hotcoffeeshake/tong-jincheng-skill)

---

## 四、phuryn/claude-usage

⭐ 0.8k｜语言：Python｜作者：phuryn

Claude Code 用户的"账单透明器"。这个本地看板能够追踪你的 **Token 使用量、费用花销与每次会话历史**，Pro/Max 订阅者还能看到精确的用量进度条。完全本地化运行，无需上传任何数据，是重度 Claude Code 用户控制成本的利器。

[项目地址](https://github.com/phuryn/claude-usage)

---

## 五、wxtsky/CodeIsland

⭐ 0.8k｜语言：Swift｜作者：wxtsky

把 AI 编程 Agent 的运行状态塞进 macOS 的**灵动岛（Dynamic Island / Notch）**——这正是 CodeIsland 在做的事。它实时展示当前 AI 编码工具（Claude、Cursor 等）的工作状态，让你在不打断心流的情况下随时掌握 Agent 动态。颜值与实用兼备，是 macOS 开发者值得一试的效率小工具。

[项目地址](https://github.com/wxtsky/CodeIsland)
