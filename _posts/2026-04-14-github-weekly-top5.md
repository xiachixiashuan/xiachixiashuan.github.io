# GitHub 周榜 Top5｜2026-04-14

本周 GitHub 涌现了一批高质量新项目，从 macOS 系统工具到 AI 自动化研究代理，从 Rust 终端组件到 Claude Code 技能扩展，多个方向同时爆发。以下是本周诞生、星标最多的 5 个新仓库速览。

## 一、momenbasel/PureMac

⭐ 900 | 语言：Swift | 作者：Mohamed Basel

PureMac 是一款完全免费、开源的 macOS 系统清理工具，定位为 CleanMyMac 的零追踪替代品。它不收集任何遥测数据，代码完全公开透明，用户可自行审计每一项清理操作的逻辑。对于在意隐私、不愿为商业清理软件付费的 Mac 用户来说，这是一个极具吸引力的选择，上线即获近千星。

[项目地址](https://github.com/momenbasel/PureMac)

## 二、vyfor/rattles

⭐ 803 | 语言：Rust | 作者：vyfor

rattles 是一个为 Rust 项目设计的极简终端 spinner 库，专注于小巧、零依赖、高度可定制。它提供了丰富的动画样式，让命令行程序的等待提示更加优雅。Rust 生态中终端 UI 组件一直是热门需求，rattles 凭借简洁的 API 和出色的性能在本周脱颖而出。

[项目地址](https://github.com/vyfor/rattles)

## 三、alchaincyf/obsidian-ai-orange-book

⭐ 548 | 语言：Markdown | 作者：alchaincyf

这是「橙皮书系列」的最新一册——将 Obsidian 与 Claude Code 深度结合，探索如何用 AI 重建个人知识管理体系（第二大脑）。书中涵盖 Obsidian 插件配置、Claude 自动化写作流、笔记图谱生成等实战场景。对 AI 效率工具感兴趣的开发者和内容创作者值得一读。

[项目地址](https://github.com/alchaincyf/obsidian-ai-orange-book)

## 四、hamen/material-3-skill

⭐ 491 | 语言：TypeScript | 作者：hamen

material-3-skill 是一个专为 Claude Code 设计的 Material Design 3 技能包，内含 30+ UI 组件实现、完整设计 token 系统和主题切换支持。开发者只需安装该技能，即可让 Claude Code 按照 Google MD3 规范生成前端代码，大幅提升 AI 辅助开发的设计一致性。

[项目地址](https://github.com/hamen/material-3-skill)

## 五、Xiangyue-Zhang/auto-deep-researcher-24x7

⭐ 387 | 语言：Python | 作者：Xiangyue Zhang

auto-deep-researcher-24x7 是一个自主 AI 研究代理，能够在你睡觉时全自动运行深度学习实验。它支持实验调度、超参数搜索、结果记录和异常自恢复，目标是让研究人员从重复性实验管理中解放出来，专注于核心创新。项目正处于活跃开发阶段，已获数百星关注。

[项目地址](https://github.com/Xiangyue-Zhang/auto-deep-researcher-24x7)
