# GitHub 周榜 Top5｜2026-04-18

本周 GitHub 热度榜再次被 AI 工具与 Agent 框架刷屏。从能「自我进化」的智能体框架，到让 Claude 代码助手拥有完美记忆的插件，再到让文档秒变 Markdown 的利器——5 个项目，覆盖 AI Agent、AI 编码、文档处理三大热点赛道，一文速览。

## 一、NousResearch/hermes-agent

⭐本周 +47.1k｜Python｜NousResearch

NousResearch 推出的新一代 AI Agent 框架，口号是「与你共同成长的 Agent」。它以 Nous Hermes 系列模型为底座，内置自适应技能树与任务规划机制，Agent 可以在执行任务过程中持续积累经验、拓展能力边界。框架支持多模型后端切换，开发者无需大幅改写代码即可接入不同 LLM。对于想搭建「越用越聪明」的 Agent 系统的团队而言，这是本周最值得关注的开源项目。

[项目地址](https://github.com/NousResearch/hermes-agent)

## 二、forrestchang/andrej-karpathy-skills

⭐本周 +42.3k｜配置文件｜forrestchang

一份精心提炼的 `CLAUDE.md` 文件，内容来源于 AI 领域传奇人物 Andrej Karpathy 对 Claude Code 使用体验的公开总结与观察。通过将这份配置文件放入项目根目录，开发者可以显著改善 Claude Code 的代码理解深度、上下文保持能力与任务执行规范性。项目零依赖，复制即用，却在一周内引爆 4 万多星，说明 AI 编码提示词工程的需求远比想象中旺盛。

[项目地址](https://github.com/forrestchang/andrej-karpathy-skills)

## 三、thedotmack/claude-mem

⭐本周 +14k｜TypeScript｜thedotmack

Claude Code 的「完美记忆」插件。它以 Claude Code 钩子（Hook）的形式运行，自动捕获每一次编码会话中 Claude 所做的决策、修改与推理过程，并结构化存储到本地。下次开启新会话时，插件会将历史上下文无缝注入，让 AI 真正「记得」你的项目约定和偏好，彻底告别每次都要重新交代背景的低效循环。

[项目地址](https://github.com/thedotmack/claude-mem)

## 四、microsoft/markitdown

⭐本周 +13.2k｜Python｜Microsoft

微软出品的通用文档转 Markdown 工具，支持 Word、Excel、PowerPoint、PDF、图片、HTML、CSV、ZIP 等几乎所有常见格式。一行命令或几行 Python 代码，即可将复杂的富文本文档转换为结构清晰的 Markdown，非常适合 RAG 数据预处理、知识库构建以及 LLM 文档摄取流水线。得益于微软官方背书与持续维护，它已成为 AI 工程师工具箱中的标配组件。

[项目地址](https://github.com/microsoft/markitdown)

## 五、multica-ai/multica

⭐本周 +10.1k｜TypeScript｜multica-ai

开源的 AI 编码 Agent 管理平台，定位是把 AI 编码 Agent 变成真正的「团队成员」。平台提供任务分配、进度追踪、并行执行与结果审核等功能，支持将复杂工程任务拆解后分发给多个 Agent 并行处理，并通过统一界面监控全局状态。对于希望将 AI 编码能力规模化落地的工程团队，这是目前开源社区最完善的多 Agent 协作方案之一。

[项目地址](https://github.com/multica-ai/multica)
