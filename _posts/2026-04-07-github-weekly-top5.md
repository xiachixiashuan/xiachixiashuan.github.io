# GitHub 周榜 Top5｜2026-04-07

本周 GitHub 涌现出一批令人眼前一亮的新项目，从 AI 编程助手到内容管理系统，覆盖了开发者日常最关心的多个领域。以下是本周精选的五个高热度项目，供你参考。

## 一、Gitlawb/openclaude

⭐ 18754 ｜ TypeScript ｜ 作者：Gitlawb

openclaude 是一款开源的编程智能体命令行工具，其核心理念是"一个接口，百模皆通"。它兼容 OpenAI、Gemini、DeepSeek、Ollama、Codex、GitHub Models 等超过 200 种模型，只要支持 OpenAI 兼容 API 的服务均可接入。对于希望摆脱单一厂商锁定、在不同大模型间灵活切换的开发者而言，这是一把难得的瑞士军刀。

[项目地址](https://github.com/Gitlawb/openclaude)

## 二、santifer/career-ops

⭐ 15990 ｜ JavaScript ｜ 作者：santifer

career-ops 将求职流程打包成一套 AI 驱动的自动化系统，内置 14 种技能模式，涵盖简历定制、面试准备、批量投递等环节，并配有 Go 语言编写的管理面板与 PDF 生成功能。它就像一位不知疲倦的求职助理，帮你把繁琐的应聘事务变成可批量处理的流水线。特别适合正处于求职期或频繁跳槽的技术人。

[项目地址](https://github.com/santifer/career-ops)

## 三、milla-jovovich/mempalace

⭐ 11910 ｜ Python ｜ 作者：milla-jovovich

mempalace 自称是目前基准测试中得分最高的 AI 记忆系统，且完全免费。它基于 ChromaDB 与 MCP 协议，为大语言模型提供持久化的长期记忆能力——让 AI 不再是每次对话都从零开始的失忆者，而是能够积累上下文、跨会话保留信息的"记性极好的助手"。对构建个人助理或知识管理类应用的开发者极具参考价值。

[项目地址](https://github.com/milla-jovovich/mempalace)

## 四、emdash-cms/emdash

⭐ 8068 ｜ TypeScript ｜ 作者：emdash-cms

EmDash 是一款基于 Astro 框架构建的全栈 TypeScript 内容管理系统，定位为 WordPress 的精神继承者。它在保留传统 CMS 易用性的同时，拥抱了现代前端生态，适合希望告别 PHP 遗产、又不想从头搭建内容系统的开发者。如果你正在为个人博客或小型媒体项目寻找一个轻量、现代的 CMS 方案，EmDash 值得一试。

[项目地址](https://github.com/emdash-cms/emdash)

## 五、HKUDS/OpenHarness

⭐ 7025 ｜ Python ｜ 作者：HKUDS

OpenHarness 是一个开放式智能体测试框架，专注于为 AI Agent 提供标准化的评测与运行环境。在 Agent 工程日益复杂的今天，如何系统地测试、比较不同 Agent 的能力成为难题，OpenHarness 正是为填补这一空白而生。对从事 Agent 研究或产品落地的团队来说，它可以作为统一的实验基础设施来使用。

[项目地址](https://github.com/HKUDS/OpenHarness)
