# GitHub 周榜 Top5｜2026-05-18

本周 GitHub 新星项目出炉！从 Facebook 的视觉几何 AI 到 Rust 编写的极简 Agent，从 Android 开发技能包到浏览器端体素岛屿游戏，五个各具特色的开源项目一次看完。

## 一、facebookresearch/vggt-omega

⭐ 611 | Python | Facebook Research

VGGT Omega 是 Facebook Research 发布的视觉几何基础模型，荣获 **CVPR 2026 Oral** 认可。该项目在视觉几何感知领域实现了重要突破，能够从图像中精确重建三维几何信息。作为 VGGT 系列的最新演进，Omega 版本在推理精度和场景泛化能力上全面提升，为机器人、自动驾驶和 AR/VR 场景提供强有力的视觉感知底座。

[项目地址](https://github.com/facebookresearch/vggt-omega)

## 二、Kappaemme-git/codex-complexity-optimizer

⭐ 625 | Python | Kappaemme-git

一个专为 AI 编程助手设计的 **Codex Skill**，专注于代码库复杂度分析与性能优化报告生成。它能安全地扫描大型代码库，识别高复杂度模块和潜在性能瓶颈，输出结构化的优化建议报告。对于维护大型工程的开发者而言，这是将 AI 编程工具与工程质量管理相结合的实用尝试。

[项目地址](https://github.com/Kappaemme-git/codex-complexity-optimizer)

## 三、chrisbanes/skills

⭐ 603 | Kotlin / Android | chrisbanes

著名 Android 开源开发者 chrisbanes 发布的 **Skills 合集**，涵盖 Kotlin、Jetpack Compose、Coroutines 及 Kotlin Multiplatform 等核心技术栈。每个 Skill 是一份凝练的最佳实践指南，涵盖跨平台组件设计、状态管理、协程使用模式等实战场景，是 Android 和 KMP 开发者的高质量参考资料。

[项目地址](https://github.com/chrisbanes/skills)

## 四、gi-dellav/zerostack

⭐ 543 | Rust | gi-dellav

用 **Rust** 编写的极简编程 Agent，以极低内存占用和高性能为核心设计目标。ZeroStack 摒弃了主流 Agent 框架的重度依赖，利用 Rust 的所有权模型和零成本抽象实现高效的代码生成与执行流程。对于追求轻量、高性能 AI 编程工具链的开发者，这是一个值得关注的实验性项目。

[项目地址](https://github.com/gi-dellav/zerostack)

## 五、boona13/mykonos-island-voxels

⭐ 557 | JavaScript | boona13

一个纯原生 JavaScript 实现的 **浏览器端等轴测体素岛屿建造器**，以希腊小岛 Mykonos 的阳光色调为设计灵感。无需任何打包工具，使用原生 ES Modules 实现，支持移动端触控操作。该项目展示了不依赖框架、仅靠 Canvas 2D API 就能实现精美交互体验的可能性，是 WebGL 替代方案的优秀示例。

[项目地址](https://github.com/boona13/mykonos-island-voxels)
