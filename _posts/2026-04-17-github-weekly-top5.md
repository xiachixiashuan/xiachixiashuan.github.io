# GitHub 周榜 Top5｜2026-04-17

过去 7 天，GitHub 诞生了数千个新项目。本期从中精选 5 个最受关注的开源仓库，涵盖 Web 终端、AI 情报、设计提取、编程语言与 PPT 生成，带你一次看懂本周开源热点。

## 一、vercel-labs/wterm

⭐ 1,036 | 语言：TypeScript | 作者：Vercel Labs

Vercel 官方实验室出品的 Web 终端模拟器，让浏览器直接运行完整的终端环境。基于现代 Web 技术构建，支持 xterm.js 协议，可无缝集成到任何 Web 应用中，是 Cloud IDE 和在线开发平台的理想基础组件。

[项目地址](https://github.com/vercel-labs/wterm)

## 二、BuilderPulse/BuilderPulse

⭐ 735 | 语言：N/A | 作者：BuilderPulse

为独立开发者和创业者打造的 AI 驱动每日情报服务，每天早晨自动聚合 10+ 信息源、回答 20 个核心问题，让你在喝咖啡的时间里掌握全球创业动态。订阅后即可获得个性化的开发者日报，极大降低信息焦虑。

[项目地址](https://github.com/BuilderPulse/BuilderPulse)

## 三、Manavarya09/design-extract

⭐ 675 | 语言：JavaScript | 作者：Manavarya09

一行命令提取任意网站的完整设计语言——颜色、字体、间距、阴影、圆角等设计 Token 全部自动化输出。提供 `npx` CLI 工具和 Claude Code 插件双入口，设计师和前端工程师的必备效率神器。

[项目地址](https://github.com/Manavarya09/design-extract)

## 四、WeaveMindAI/weft

⭐ 659 | 语言：Rust | 作者：WeaveMindAI

专为 AI 系统设计的新编程语言，使用 Rust 实现高性能运行时。Weft 针对大模型推理、Agent 编排和 AI 工作流场景进行了深度优化，语法简洁，类型安全，旨在成为 AI 原生时代的系统级语言。

[项目地址](https://github.com/WeaveMindAI/weft)

## 五、lewislulu/html-ppt-skill

⭐ 596 | 语言：HTML | 作者：lewislulu

功能丰富的 HTML PPT 生成 Agent Skill，内置 24 套主题、31 种布局和 20+ 种动画效果，可直接在 Claude Code 中调用生成专业级 HTML 演示文稿。无需安装 Office，纯浏览器渲染，分享即用。

[项目地址](https://github.com/lewislulu/html-ppt-skill)
