# GitHub 周榜 Top5｜2026-04-23

本周 GitHub 涌现出一批高质量开源项目，涵盖 AI 设计工具、营销数字孪生、智能 Agent 框架、浏览器扩展等多个方向。以下是过去 7 天内新创建、快速收获星标的 5 个项目速览。

## 一、OpenCoworkAI/open-codesign

⭐ 1.2k｜TypeScript｜作者：OpenCoworkAI

Claude Design 的开源替代品。只需一键导入你的 Claude Code / Codex API 密钥，即可通过自然语言 Prompt 生成原型、幻灯片和 PDF。支持多模型（Claude、GPT、Gemini、Kimi、GLM、Ollama），完全本地优先、BYOK、MIT 协议。对于不想订阅 Claude Max 的开发者来说，这是一个极具吸引力的 Design-to-Code 工作台。

[项目地址](https://github.com/OpenCoworkAI/open-codesign)

## 二、432539/gpt2api

⭐ 794｜Go｜作者：432539

基于 chatgpt.com 逆向工程构建的 OpenAI 兼容 SaaS 网关。支持 GPT-IMAGE-2 批量出图、多账号池调度、高并发与积分计费。用 Go 编写，性能出色，适合需要低成本批量调用图像生成能力的开发者和中小团队自建 API 中转服务。

[项目地址](https://github.com/432539/gpt2api)

## 三、OranAi-Ltd/oransim

⭐ 556｜Python｜作者：OranAi-Ltd

面向营销决策的因果数字孪生平台。核心思路是：在你花出第一分钱之前，先用 AI 模拟预测每一个营销决策的效果。基于因果推断而非纯统计相关，能够给出更可解释的预测结果，适合品牌投放团队在预算有限时做 A/B 方案推演。

[项目地址](https://github.com/OranAi-Ltd/oransim)

## 四、Ademking/MD-This-Page

⭐ 500｜TypeScript｜作者：Ademking

一键将任意网页转换为干净可读的 Markdown 的浏览器扩展，支持 Chrome 和 Firefox。对于需要将网页内容喂给 LLM、整理到笔记软件或做文档存档的用户来说，这个扩展极大简化了工作流。无需服务器，完全在本地处理。

[项目地址](https://github.com/Ademking/MD-This-Page)

## 五、cosmicstack-labs/mercury-agent

⭐ 479｜TypeScript｜作者：cosmicstack-labs

一款"以灵魂为驱动"的 AI Agent 框架，内置权限硬化工具集、Token 预算控制和多渠道接入（CLI / Telegram）。设计目标是 24/7 全天候运行，兼顾安全性与自主性。支持自定义工具、多模型接入，适合构建需要长期运行的个人助理或自动化 Bot。

[项目地址](https://github.com/cosmicstack-labs/mercury-agent)
