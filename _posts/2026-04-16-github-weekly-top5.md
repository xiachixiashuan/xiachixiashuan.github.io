# GitHub 周榜 Top5｜2026-04-16

本周 GitHub 又涌现出一批亮眼新项目，从 AI 辅助协议逆向、预测市场机器人，到 GEO 内容生产系统、流量防检测工具与 Android 应用冻结管理器，覆盖开发效率、安全研究与移动端工具多条赛道。以下是过去 7 天涨星最快的 5 个全新开源仓库。

## 一、Mouseww/anything-analyzer

⭐ 796 | 语言：TypeScript | 作者：Mouseww

一款「傻瓜式」协议分析工具，结合浏览器调试协议（CDP）捕获与 AI 分析能力，可自动为注册机、第三方 API 接入或任意浏览器/客户端行为生成可读性强的协议文档。无需手动抓包或逐字节解析，大幅降低逆向工程门槛。对于需要对接各类平台 API 的开发者而言，它像一位「永不疲倦的协议翻译官」。

[项目地址](https://github.com/Mouseww/anything-analyzer)

## 二、sterlingcrispin/nothing-ever-happens

⭐ 771 | 语言：Python | 作者：sterlingcrispin

一个 Polymarket 预测市场机器人，策略简单粗暴：对所有非体育类事件下注「No（不会发生）」。名字本身就是它的投资逻辑——大多数"惊天大事"最终往往不了了之。项目代码轻量，提供了一种趣味性视角审视预测市场中的偏见与博弈，也可作为量化策略研究的起点。

[项目地址](https://github.com/sterlingcrispin/nothing-ever-happens)

## 三、yaojingang/GEOFlow

⭐ 673 | 语言：PHP | 作者：yaojingang

面向 GEO（生成引擎优化）场景的开源内容生产系统，内置 AI 任务调度、多级审核工作流与多平台发布能力。随着 AI 搜索引擎成为流量新入口，GEOFlow 帮助内容团队系统化地生产「对 AI 友好」的结构化内容，是 SEO 向 GEO 时代迁移的实用工程化方案。

[项目地址](https://github.com/yaojingang/GEOFlow)

## 四、patterniha/SNI-Spoofing

⭐ 603 | 语言：Python | 作者：patterniha

通过修改 IP/TCP 头部字段来绕过深度包检测（DPI）的研究工具。SNI（服务器名称指示）是 TLS 握手中暴露目标域名的关键字段，本项目探索了在不破坏连接的前提下扰乱 DPI 识别的技术路径。适合网络安全研究者、CTF 参与者以及对流量混淆感兴趣的开发者学习参考。

[项目地址](https://github.com/patterniha/SNI-Spoofing)

## 五、sogonov/anubis

⭐ 602 | 语言：Kotlin | 作者：sogonov

Android 端应用管理工具，核心功能是根据 VPN 连接状态自动冻结或解冻应用分组。以古埃及守护神"阿努比斯"命名，寓意对应用生死的掌控。对于需要在工作/娱乐模式间快速切换、或希望在断开 VPN 时自动隐藏敏感应用的用户，这是一个优雅的自动化解决方案。

[项目地址](https://github.com/sogonov/anubis)
