# GitHub 周榜 Top5｜2026-04-22

本周 GitHub 再度涌现一批亮眼新项目，涵盖 AI 设计工具、内容排版、PDF 编辑器、微信数据可视化等多个方向，7 天内累计斩获数千 Star。以下是本期精选五个值得关注的开源项目。

## 一、alchaincyf/huashu-design

⭐ 3053 | 语言：HTML | 作者：alchaincyf

花树设计（Huashu Design）是专为 Claude Code 打造的 HTML 原生设计 Skill，让 AI Agent 直接生成高保真原型、幻灯片与动态动画，内置 20 条设计哲学与 5 维设计评审框架，并支持导出为 MP4。**完全无框架依赖（Agent-agnostic），开箱即用。** 无论是产品经理出原型还是工程师展示方案，都能大幅提升效率。

[项目地址](https://github.com/alchaincyf/huashu-design)

## 二、tw93/Kami

⭐ 1583 | 语言：HTML | 作者：tw93

"Good content deserves good paper."——Kami 的口号直接点明了它的定位：为优质内容提供优雅的「纸张」。这是一款聚焦内容呈现与排版美化的工具，让文字以更精致的形式输出，适合博客写作、内容创作者以及追求文档品质的开发者。**仅用纯 HTML 实现，轻量且无任何外部依赖。**

[项目地址](https://github.com/tw93/Kami)

## 三、SteveTheKiller/KillerPDF

⭐ 639 | 语言：C# | 作者：SteveTheKiller

KillerPDF 是一款面向 Windows 的便携式 PDF 编辑器，基于 WPF 构建，遵循 GPLv3 协议。**无需安装、无需注册账号、零订阅、零遥测**，直接解压即用。对于厌倦了臃肿商业软件的用户来说，这是处理 PDF 文件的简洁替代方案，适合在办公或个人设备上快速部署。

[项目地址](https://github.com/SteveTheKiller/KillerPDF)

## 四、ZeroZ-lab/cc-design

⭐ 563 | 语言：JavaScript | 作者：ZeroZ-lab

cc-design 是为 AI Agent 量身打造的高保真 HTML 设计与原型指导 Skill，与 huashu-design 方向相近但定位不同——它更侧重于**为通用 AI 编程助手提供系统化的设计提示与 HTML 输出规范**，帮助模型生成更接近真实产品品质的前端界面。适合希望在 Agentic 流程中加入设计能力的开发者。

[项目地址](https://github.com/ZeroZ-lab/cc-design)

## 五、zhuyansen/wx-favorites-report

⭐ 532 | 语言：Python | 作者：zhuyansen

微信收藏可视化工具，作为 Claude Code Skill 运行，实现从加密本地数据库到交互式 HTML 报告的端到端管线。**无需破解微信服务器，直接读取设备本地的加密 SQLite 文件**，生成可搜索、可统计的收藏内容仪表盘。对有大量微信收藏待整理的用户颇具吸引力。

[项目地址](https://github.com/zhuyansen/wx-favorites-report)
