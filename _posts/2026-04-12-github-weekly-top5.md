# GitHub 周榜 Top5｜2026-04-12

本周 GitHub 又涌现出一批高关注度的新项目，涵盖 AI 图表生成、LLM 知识库、Claude Code 技能、人格蒸馏 Skill 合集等多个热门方向。以下是过去 7 天内诞生的 5 个新星项目，带你快速掌握开源社区最新动态。

## 一、yizhiyanhua-ai/fireworks-tech-graph

⭐ 967 | 语言：Shell | 作者：yizhiyanhua-ai

这是一个专为 Claude Code 设计的技术图表生成 Skill，可自动产出生产级 SVG+PNG 格式的技术架构图。支持 8 种图表类型（流程图、时序图、架构图等）和 5 种视觉风格，内置深度的 AI/Agent 领域知识，一行指令即可将复杂系统架构可视化，是工程文档自动化的利器。

[项目地址](https://github.com/yizhiyanhua-ai/fireworks-tech-graph)

## 二、adamlyttleapps/claude-skill-app-onboarding-questionnaire

⭐ 740 | 语言：- | 作者：adamlyttleapps

这是一个 Claude Code Skill，专门用于设计和构建高转化率的问卷式 App 引导流程。它参考了 Mob、Headspace、Noom 等头部订阅类 App 的成熟转化模式，让开发者快速生成符合最佳实践的用户引导体验，有效提升留存率与付费转化率，适合独立开发者和产品团队使用。

[项目地址](https://github.com/adamlyttleapps/claude-skill-app-onboarding-questionnaire)

## 三、nashsu/llm_wiki

⭐ 718 | 语言：TypeScript | 作者：nashsu

LLM Wiki 是一款跨平台桌面应用，能将你的文档自动整理为结构化的互联知识库。与传统 RAG 每次从头检索不同，它通过 LLM 对文档进行增量式解析，逐步构建并维护持久化的 Wiki 图谱，让知识真正"活"起来。无论是个人笔记还是团队文档，都能一键转化为可检索、可关联的智能知识库。

[项目地址](https://github.com/nashsu/llm_wiki)

## 四、tmstack/awesome-persona-skills

⭐ 664 | 语言：- | 作者：tmstack

一个围绕"人格蒸馏"理念整理的 Agent Skill 合集，收录了同事、老板、前任、自己、永生、女娲等多种角色扮演技能。每个 Skill 可让 AI 深度模拟特定人格视角，适合情感陪伴、角色创作、心理探索等多种应用场景，是 Claude Code Skill 生态中颇具创意的一个方向。

[项目地址](https://github.com/tmstack/awesome-persona-skills)

## 五、QLHazyCoder/codex-oauth-automation-extension

⭐ 651 | 语言：JavaScript | 作者：QLHazyCoder

一款 Chrome 扩展，支持 OpenAI OAuth 注册自动化，涵盖验证码获取、CPA 回调验证与自动恢复等完整流程。对于需要在研究或合规测试场景下管理多个 OpenAI 账号的开发者，这款扩展大幅简化了繁琐的注册与验证操作，节省大量重复性工作时间。

[项目地址](https://github.com/QLHazyCoder/codex-oauth-automation-extension)
