# GitHub 周榜 Top5｜2026-05-16

本周 GitHub 新仓库中诞生了不少亮眼项目，涵盖安全研究、AI 工具、桌面应用与交互创作，以下是过去 7 天内新建且获得最多星标的 5 个仓库速览。

## 一、Nightmare-Eclipse/YellowKey

⭐ 2.4k｜语言：N/A｜作者：Nightmare-Eclipse

YellowKey 是一份关于 Windows BitLocker 绕过漏洞的安全研究披露，详细描述了磁盘加密机制中的潜在攻击面。该项目短时间内在安全社区引发广泛讨论，被大量渗透测试研究人员收藏。它提醒系统管理员及时跟进安全补丁，同时为安全研究员提供了分析参考材料，是本周安全领域的热门披露。

[项目地址](https://github.com/Nightmare-Eclipse/YellowKey)

## 二、nexu-io/html-anything

⭐ 2.0k｜语言：HTML｜作者：nexu-io

html-anything 是一款面向 AI Agent 的本地 HTML 编辑器，内置 75 个技能模板，支持杂志、幻灯片、海报、小红书 / 推文、原型图、数据报告、Hyperframes 等 9 种页面形式输出。提供沙箱预览环境，一键导出至微信、X、知乎或本地 HTML / PNG，零 API Key 依赖，兼容 Claude Code、Cursor、Codex、Gemini、Copilot、OpenCode 等主流 AI 编程工具接入。

[项目地址](https://github.com/nexu-io/html-anything)

## 三、yetone/native-feel-skill

⭐ 1.1k｜语言：N/A｜作者：yetone

native-feel-skill 是一套跨平台桌面应用原生感设计 Skill，从对 Raycast 2.0 的深度逆向工程中提炼出 8 条架构原则与 4 层设计体系，并配备 WebKit / WebView2 兼容生存指南和 75 项上线审查清单。适合希望在 Web 技术栈上打造原生质感 UI 的开发者，提供可直接复用的设计决策框架。

[项目地址](https://github.com/yetone/native-feel-skill)

## 四、jasonkneen/tiny-world-builder

⭐ 540｜语言：HTML｜作者：jasonkneen

tiny-world-builder 是一个轻量级互动世界构建器，使用纯 HTML 实现，可在浏览器中直接运行，无需任何外部依赖。用户可通过简洁界面快速搭建并探索微型世界，交互友好，适合原型快速验证和创意实验。项目代码简洁，也是学习 HTML Canvas / DOM 交互的好示例。

[项目地址](https://github.com/jasonkneen/tiny-world-builder)

## 五、lucasfrre/BongoCat-Desktop

⭐ 413｜语言：C++｜作者：lucasfrre

BongoCat-Desktop 是功能丰富的直播桌面叠层工具，以 Bongo Cat 角色为主角。支持鼠标与键盘轨迹实时跟踪、绘图板笔压感应、osu! 游玩模式切换，内置自定义皮肤编辑器、透明背景渲染、OBS Studio 插件、手柄支持及 RGB 灯光同步。是直播主和游戏创作者的实用桌面宠物工具，也适合键位可视化展示需求。

[项目地址](https://github.com/lucasfrre/BongoCat-Desktop)
