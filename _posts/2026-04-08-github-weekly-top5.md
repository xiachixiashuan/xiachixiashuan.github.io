# GitHub 周榜 Top5｜2026-04-08

本周 GitHub 新冒出一批与 AI 编程助手相关的热门项目，以下是本周新上榜的五个值得关注的开源仓库。

## 一、safishamsi/graphify

⭐ 6.9k ｜ Python ｜ 作者：safishamsi

Graphify 是一个 AI 编程助手技能扩展，兼容 Claude Code、Codex、OpenCode 等主流工具。它能将任意代码目录、文档、论文或图片转化为可查询的知识图谱，让 AI 助手在回答问题前先理解整个项目的语义结构，而非盲目检索文件。特别适合在大型单仓库（monorepo）中工作、需要跨文件理解依赖关系的开发者。

[项目地址](https://github.com/safishamsi/graphify)

## 二、ultraworkers/claw-code-parity

⭐ 6.6k ｜ Rust ｜ 作者：ultraworkers

这是 claw-code 的 Rust 移植版，专门用于在原仓库迁移期间维持功能等价性。如果你对用 Rust 重写 AI 编程终端工具的工程权衡感兴趣，或者正在等待 claw-code 正式发布，这个仓库记录了移植过程中最值得参考的实现细节。

[项目地址](https://github.com/ultraworkers/claw-code-parity)

## 三、JuliusBrussee/caveman

⭐ 5.9k ｜ Python ｜ 作者：JuliusBrussee

"为什么用很多词，用少词就行。" Caveman 是一个 Claude Code 技能扩展，通过让 AI 使用极简的原始人语法（caveman speak）来表达思路，将实际消耗的 token 数量压缩约 65%。对于需要长时间运行 AI 对话、想控制 API 成本的开发者来说，这是一个思路独特的节省方案。

[项目地址](https://github.com/JuliusBrussee/caveman)

## 四、kevinrgu/autoagent

⭐ 3.8k ｜ Python ｜ 作者：kevinrgu

Autoagent 专注于自主的 agent 工程脚手架，目标是让开发者无需手动干预，AI 即可完成复杂的多步骤编程任务。项目目前偏向实验性，但已积累了相当的关注度，适合对 agent 自主化工作流感兴趣的研究者和工程师。

[项目地址](https://github.com/kevinrgu/autoagent)

## 五、alchaincyf/nuwa-skill

⭐ 2.5k ｜ 作者：alchaincyf

女娲.skill 是一个 AI 技能文件，允许用户从任意人物的公开内容中提炼其思维模式——包括心智模型、决策启发式和表达风格——生成可供 Claude 等 AI 工具复用的技能配置。项目作者将其定位为"蒸馏下一个员工"的工具，适合希望将特定专家的决策框架注入 AI 助手的团队。

[项目地址](https://github.com/alchaincyf/nuwa-skill)
