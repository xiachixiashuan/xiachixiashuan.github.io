# GitHub 周榜 Top5｜2026-04-21

本周 GitHub 涌现了一批令人眼前一亮的新项目，从极简图表工具到跨平台 Agent 配置框架，从 GPT 图像提示词库到公开 BTC 实盘数据，每一个都值得开发者和 AI 爱好者关注。以下五个项目均于过去 7 天内创建，按 Star 数排列。

## 一、cathrynlavery/diagram-design

⭐ 961 | 语言：HTML | 作者：cathrynlavery

专为 Claude Code 设计的 13 种编辑级图表类型，纯 HTML + SVG 实现，零外部依赖。彻底告别冗余阴影和 Mermaid 生成的低质图表，每种图表都自包含、可直接嵌入文档或报告。对于需要在 AI 辅助开发流程中快速生成专业示意图的工程师来说，这是一个即取即用的高质量素材库。

[项目地址](https://github.com/cathrynlavery/diagram-design)

## 二、EvoLinkAI/awesome-gpt-image-2-prompts

⭐ 838 | 语言：N/A | 作者：EvoLinkAI

精心整理的 GPT-Image-2 提示词与生成示例合集，覆盖人像、海报、UI 线框图、角色设定图及社区实验等多个场景。每条提示词附有对应生成效果，方便对比学习。随着 GPT-Image-2 能力持续提升，这类高质量提示词库正在成为 AI 图像生成领域的稀缺资源。

[项目地址](https://github.com/EvoLinkAI/awesome-gpt-image-2-prompts)

## 三、bwjoke/BTC-Trading-Since-2020

⭐ 782 | 语言：N/A | 作者：bwjoke

作者将自 2020 年起的比特币实盘交易记录全部公开，包含完整的交易上下文信息。这份数据对量化研究者、策略回测开发者及加密货币分析师具有极高价值。公开真实交易数据的勇气本身也令人钦佩，社区已开始围绕这份数据展开多维度分析。

[项目地址](https://github.com/bwjoke/BTC-Trading-Since-2020)

## 四、codejunkie99/agentic-stack

⭐ 760 | 语言：Python | 作者：codejunkie99

一个可移植的 `.agent/` 文件夹方案，内含记忆、技能和协议配置，可无缝接入 Claude Code、Cursor、Windsurf、OpenCode、OpenClaw、Hermes 或自定义 Python 项目。核心理念是"一份配置，多工具通用"——切换 AI 工具时知识库和技能不丢失，真正实现 Agent 能力的跨平台复用。

[项目地址](https://github.com/codejunkie99/agentic-stack)

## 五、VoltAgent/awesome-claude-design

⭐ 724 | 语言：N/A | 作者：VoltAgent

收录 68 个即用型 DESIGN.md 格式的设计系统灵感模板，覆盖多种 UI 风格。只需将其中一个文件放入项目根目录，Claude Code 等 AI 工具即可据此在单次对话中搭建完整 UI 框架，大幅缩短从创意到原型的时间。

[项目地址](https://github.com/VoltAgent/awesome-claude-design)
