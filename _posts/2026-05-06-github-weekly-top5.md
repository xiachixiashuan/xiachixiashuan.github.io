# GitHub 周榜 Top5｜2026-05-06

本周 GitHub 涌现出一批高质量新项目，从 AI 安全测试到 LLM 从零构建，从系统诊断工具到世界模型研究框架，多个方向同时爆发。以下是本周新晋 Star 增速最快的 5 个项目速览。

## 一、vercel-labs/deepsec

⭐ 1062 ｜ TypeScript ｜ Vercel Labs

Deepsec 是一个由编程 Agent 驱动的**代码安全漏洞扫描框架**，专为开发团队提供自动化安全审计能力。它将 AI 编程 Agent 引入安全测试流程，能主动探测代码库中的漏洞，输出结构化报告，无需手动编写每一条测试规则。面向希望将安全左移融入 CI/CD 的工程团队，极大降低了安全测试的门槛。

[项目地址](https://github.com/vercel-labs/deepsec)

## 二、raiyanyahya/how-to-train-your-gpt

⭐ 445 ｜ Jupyter Notebook ｜ Raiyan Yahya

这是一个"从零构建现代 LLM"的教学级项目，**每行代码都有注释，像给五岁孩子讲一样**通俗易懂。作者覆盖了 Transformer 架构、注意力机制、预训练与微调的完整流程，配以 Jupyter Notebook 让读者可以边看边跑，是目前少有的兼顾深度与可读性的 LLM 实战教程。

[项目地址](https://github.com/raiyanyahya/how-to-train-your-gpt)

## 三、crafter-station/petdex

⭐ 369 ｜ TypeScript ｜ Crafter Station

Petdex 是一个 Codex 宠物的**公开展示画廊**，收录来自社区的各种动态像素宠物形象。Codex 宠物是 OpenAI Codex 编辑器中的虚拟伴侣，Petdex 让开发者可以分享、发现和收藏这些充满个性的小角色，形成了一个轻松有趣的开发者社区文化项目。

[项目地址](https://github.com/crafter-station/petdex)

## 四、matthart1983/syswatch

⭐ 358 ｜ Rust ｜ Matt Hart

Syswatch 是一款**单主机系统诊断 TUI 工具**，基于 Rust 构建，提供十二个标签页的系统信息视图，包含 CPU、内存、磁盘、网络、进程等维度，全部以平实英语展示洞察，无需专业运维背景也能快速定位问题。它是同作者 netwatch 的兄弟项目，延续了简洁而功能强大的设计哲学。

[项目地址](https://github.com/matthart1983/syswatch)

## 五、simchowitzlabpublic/nano-world-model

⭐ 353 ｜ Python ｜ Simchowitz Lab

Nano World Model 是一个**极简主义世界模型研究仓库**，"自带电池"开箱即用，专为推进世界模型科学设计。项目以最小化配置提供完整的世界模型训练与评测管线，让研究者聚焦在核心算法创新而非工程搭建，适合 AI 研究人员快速复现和对比不同世界模型方案。

[项目地址](https://github.com/simchowitzlabpublic/nano-world-model)
