# GitHub 周榜 Top5｜2026-04-15

过去七天，GitHub 上又涌现出一批令人眼前一亮的新项目。本期精选 5 个星标增速最快的新仓库，覆盖 AI 工具、语音合成、开发者效率、网络工具与 macOS 应用，一文速览本周开源热点。

## 一、AgentSeal/codeburn

⭐ 874 ｜ TypeScript ｜ AgentSeal

你知道自己用 Claude Code 或 Codex 每天烧了多少钱吗？**codeburn** 是一个交互式 TUI 仪表板，专门用来可视化 AI 编码工具的 Token 消耗与费用开销。它实时抓取本地日志，以终端图表的形式呈现每次会话的成本分布，帮助开发者精准掌控 AI 助手的使用效率。对重度使用 Claude Code 或 OpenAI Codex 的工程师来说，这是一个"看得见"的成本监控神器。

[项目地址](https://github.com/AgentSeal/codeburn)

## 二、OpenMOSS/MOSS-TTS-Nano

⭐ 736 ｜ Python ｜ MOSI.AI & OpenMOSS

**MOSS-TTS-Nano** 是来自复旦 OpenMOSS 团队的开源多语言极简语音合成模型。"Nano"之名实至名归——模型体积极小，却支持多种语言的流畅 TTS 生成。它面向端侧部署和低资源场景设计，无需云端 API 即可本地运行。对于需要在应用中集成轻量语音合成的开发者，这是目前开源生态中难得的精品选择。

[项目地址](https://github.com/OpenMOSS/MOSS-TTS-Nano)

## 三、alchaincyf/darwin-skill

⭐ 572 ｜ HTML ｜ alchaincyf

受 Autoresearch 思路启发，**darwin-skill** 是一套让 Claude Skill 自主进化的系统。它的工作流程是：评估现有 Skill → 生成改进方案 → 自动测试 → 保留更优版本或回滚。整个过程无需人工干预，Skill 像生物进化一样在迭代中变得更强。这是目前最具创意的 Claude 生态项目之一，为 AI Skill 工程化提供了全新思路。

[项目地址](https://github.com/alchaincyf/darwin-skill)

## 四、retlehs/quien

⭐ 520 ｜ Go ｜ retlehs

**quien**（西班牙语"谁"）是用 Go 编写的新一代 WHOIS 查询工具。相比系统自带的 `whois` 命令，它提供更清晰的结构化输出、更好的格式解析，以及对多种 TLD 的广泛支持。对于需要频繁做域名溯源、安全排查或运维调试的工程师，quien 是一个值得加入工具箱的小巧利器。

[项目地址](https://github.com/retlehs/quien)

## 五、lzhgus/Capso

⭐ 377 ｜ Swift ｜ lzhgus

**Capso** 是 macOS 上的开源截图与录屏工具，定位为 CleanShot X 的免费原生替代品。它基于 Swift 原生开发，无需订阅，支持截图标注、区域录屏等主流功能，界面简洁、性能流畅。对于不想为截图工具付月费、又希望拥有专业功能的 Mac 用户，Capso 是目前最值得关注的开源选择。

[项目地址](https://github.com/lzhgus/Capso)
