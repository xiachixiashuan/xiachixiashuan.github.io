# GitHub 周榜 Top5｜2026-04-09

本周 GitHub 又诞生了一批令人眼前一亮的新项目，涵盖 AI 工具、效率神器、底层性能库等多个领域。以下是过去 7 天内斩获最多星标的 5 个新项目，助你快速掌握开源社区最新动态。

## 一、farzaa/clicky

⭐ 2.2k｜语言：Swift｜作者：farzaa

Clicky 是一款运行在 Mac 上的 AI 教学助手，悄悄地"住"在你的光标旁边。它能实时看到你的屏幕内容，和你对话，甚至能像真人老师一样指向屏幕上的具体元素，全程免费。项目基于 Claude API + ScreenCaptureKit 构建，开源版本支持自行接入 Anthropic、AssemblyAI 和 ElevenLabs API key，通过 Cloudflare Worker 代理保护密钥安全。**作者在 X 上发布的演示视频迅速走红，已积累大量用户。**

[项目地址](https://github.com/farzaa/clicky)

## 二、GitFrog1111/badclaude

⭐ 1.5k｜语言：HTML/JavaScript｜作者：GitFrog1111

一个以幽默方式解决"AI 卡顿焦虑"的小工具：当 Claude Code 执行太慢时，用鞭子"抽"一下它！点击托盘图标生成虚拟鞭子，点击即可触发，程序会发送 Ctrl-C 中断信号并附上 5 种鼓励性消息。**项目路线图中还计划记录"抽打次数"——当机器人接管世界时，可以根据这份数据为人类排个顺序。** 轻松搞笑，却也戳中了不少开发者的痛点。

[项目地址](https://github.com/GitFrog1111/badclaude)

## 三、LaurieWired/tailslayer

⭐ 1.4k｜语言：C++｜作者：LaurieWired

Tailslayer 是一个专注于降低内存读取尾延迟（tail latency）的 C++ 库。尾延迟是性能优化中最难啃的骨头——P99 响应时间往往比均值高出数倍，严重影响用户体验。该库通过预取策略和内存访问模式优化，有效削平延迟曲线的长尾。**对于高性能数据库、游戏引擎、实时系统等场景，这类底层优化工具具有重要价值。**

[项目地址](https://github.com/LaurieWired/tailslayer)

## 四、afar1/fieldtheory-cli

⭐ 1.3k｜语言：TypeScript｜作者：afar1

一款面向 Mac 用户的命令行工具，可将 X（原 Twitter）上的所有收藏书签同步并本地化存储。完全免费开源，无需订阅任何服务。对于习惯用 Twitter 收藏文章、代码片段的开发者来说，**数据本地化意味着随时可以全文检索、离线访问，不再受制于平台的书签功能限制。** TypeScript 编写，扩展性强，社区已在讨论添加全文搜索和 Obsidian 导出等功能。

[项目地址](https://github.com/afar1/fieldtheory-cli)

## 五、KKKKhazix/khazix-skills

⭐ 1.2k｜语言：Markdown/YAML｜作者：KKKKhazix

由"数字生命卡兹克"开源的 AI Skills 合集，涵盖多种实用的 Claude Code 技能包。随着 AI 编程助手进入"可编程"时代，自定义 Skill 成为提升工作流效率的关键——**不同于单纯的提示词，Skill 可以封装完整的工作流逻辑，实现真正意义上的自动化。** 这个合集汇聚了作者在实际使用中沉淀下来的高质量技能模板，对想要系统化构建 AI 工作流的开发者颇具参考价值。

[项目地址](https://github.com/KKKKhazix/khazix-skills)
