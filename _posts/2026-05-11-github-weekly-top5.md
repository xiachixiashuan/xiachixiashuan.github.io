# GitHub 周榜 Top5｜2026-05-11

本周 GitHub 新星涌现，从 AI 驱动的 3D 细胞探索工具，到让 Codex 用 DeepSeek 降本提效的黑科技，再到原生 macOS 视频下载利器，每一个项目都代表着开发者社区的最新创造力。以下是本周诞生不足 7 天、已跻身热榜的 5 个项目。

## 一、huangserva/3DCellForge

⭐ 738 ｜语言：JavaScript ｜作者：huangserva

3DCellForge 是一款 AI 驱动的交互式 3D 细胞生成与探索工作台，让用户通过自然语言描述即可实时构建、旋转、剖析三维细胞模型。它将生物可视化与生成式 AI 结合，无论是科学教育还是研究演示，都能迅速产出高质量的 3D 场景。项目基于 WebGL 渲染，无需安装任何本地依赖，打开浏览器即可上手。在生物信息学与 AI 融合加速的当下，这类工具正在降低专业可视化的门槛。

[项目地址](https://github.com/huangserva/3DCellForge)

## 二、BigPizzaV3/CodexPlusPlus

⭐ 691 ｜语言：Python ｜作者：BigPizzaV3

CodexPlusPlus 是 OpenAI Codex 应用的功能增强工具，在原有 Codex 能力之上叠加了批量任务调度、上下文管理优化和多模型路由等扩展能力。它以插件式架构嵌入现有工作流，让开发者无需修改核心代码即可享受增强体验。项目提供完整的 CLI 接口与 Python API，适合需要深度定制 AI 编码助手的团队使用。在 AI 编码工具竞争日趋激烈的背景下，这类"套壳增强"方向正成为开发者社区的新热点。

[项目地址](https://github.com/BigPizzaV3/CodexPlusPlus)

## 三、mykolaharmash/notelet

⭐ 398 ｜语言：Swift ｜作者：mykolaharmash

Notelet 是一个 SwiftUI 组件，专为在 macOS/iOS 应用内展示富文本版本更新日志而设计。与普通纯文本 Changelog 不同，它支持 Markdown 渲染、版本分组、动画过渡和深色模式，开箱即用只需几行代码。对于独立开发者而言，精致的版本说明页面往往是提升用户留存的细节利器。该组件已发布到 Swift Package Manager，集成成本极低。

[项目地址](https://github.com/mykolaharmash/notelet)

## 四、fendouai/CodexSaver

⭐ 397 ｜语言：Python ｜作者：fendouai

CodexSaver 的核心思路是：用 DeepSeek 替换 Codex 在低复杂度任务中的推理调用，从而在不损失代码质量的前提下大幅降低 API 成本。它通过任务复杂度分类器动态决策路由，简单补全走 DeepSeek，复杂生成仍走 Codex，实现智能降本。项目配有详细的成本对比基准，据作者测试综合费用可降低 60% 以上。随着 AI 编程工具用量爆发，成本优化正成为企业级采用的关键障碍，CodexSaver 的思路值得关注。

[项目地址](https://github.com/fendouai/CodexSaver)

## 五、pixel-point/media-downloader

⭐ 376 ｜语言：Swift ｜作者：pixel-point

Media Downloader 是一款精心打磨的原生 macOS 视频下载应用，集下载与剪辑于一体，支持直接在应用内完成视频的下载、预览和时间轴裁剪，无需借助 ffmpeg 命令行。界面设计遵循 Apple HIG 规范，动效流畅，并在菜单栏提供快捷入口。对于经常需要下载在线视频素材的创作者和开发者，这款工具填补了 macOS 生态下缺乏优雅原生下载器的空白。

[项目地址](https://github.com/pixel-point/media-downloader)
