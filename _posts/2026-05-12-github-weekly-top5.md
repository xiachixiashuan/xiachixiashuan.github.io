# GitHub 周榜 Top5｜2026-05-12

本周 GitHub 涌现出一批高质量新项目，涵盖 AI 工具链、存储 SDK、浏览器插件等多个方向。以下五个项目均诞生于过去 7 天，在社区中迅速获得关注。

## 一、alchaincyf/huashu-md-html

⭐ 435｜CSS｜作者：alchaincyf

一站式 Markdown / HTML 双向流水线工具，封装了 markitdown、Pandoc、html-to-markdown 等库，做到"万物→Markdown、Markdown→精美 HTML、HTML→Markdown"三种能力无缝切换。对于需要在文档格式之间频繁转换的写作者和开发者来说，这是一把非常顺手的瑞士军刀。支持本地部署，CSS 主题可自定义，适合搭配任意静态博客使用。

[项目地址](https://github.com/alchaincyf/huashu-md-html)

## 二、darkrishabh/agent-skills-eval

⭐ 425｜TypeScript｜作者：darkrishabh

专为 agentskills.io 风格的 AI Agent 技能设计的测试运行器，支持 YAML 定义测试用例、JSONL 格式输出评估结果，兼容所有 OpenAI-compatible API。在 Agent 能力迭代越来越快的今天，有一套标准化的评估框架至关重要——这个项目就是在填补这个空白。CLI 驱动，可无缝集成 CI/CD 流水线。

[项目地址](https://github.com/darkrishabh/agent-skills-eval)

## 三、haydenbleasel/files-sdk

⭐ 338｜TypeScript｜作者：haydenbleasel

统一对象存储 SDK，用一套简洁的 Web Standards API 抹平了 AWS S3、Cloudflare R2、Google Cloud Storage、MinIO、Vercel Blob 等多个后端的差异。接口设计遵循 Web 原生标准，对 TypeScript 友好，几行代码即可切换存储供应商。对于需要多云部署或想降低供应商锁定风险的项目而言，是个值得关注的轻量方案。

[项目地址](https://github.com/haydenbleasel/files-sdk)

## 四、setube/stackprism

⭐ 247｜TypeScript｜作者：setube

一款浏览器插件（支持 Chrome 和 Edge，Manifest V3），可检测当前网页所使用的技术栈，涵盖前端框架、CMS、CDN、广告系统、SaaS 工具、安全组件等多个维度。对竞品分析、安全研究、技术选型调研都极为实用。界面简洁，检测结果一目了然，适合前端工程师和技术侦探日常使用。

[项目地址](https://github.com/setube/stackprism)

## 五、opensquilla/opensquilla

⭐ 221｜Python｜作者：opensquilla

主打 Token 高效的 AI Agent 框架，声称在相同预算下实现更高智能密度。支持 MCP（Model Context Protocol）、长期记忆、多 Agent 协作，基于 Python 构建，接口与主流 LLM 兼容。随着 Agent 调用成本日益受到关注，这类"省 Token、不省脑力"的框架正成为开发者工具链的重要组成部分。

[项目地址](https://github.com/opensquilla/opensquilla)
