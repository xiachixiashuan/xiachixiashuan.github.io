# GitHub 周榜 Top5｜2026-04-28

本周 GitHub 涌现出一批专注于 AI 工具与开发效率的新项目，从 PPT 提示词工程、图片生成技能到 AI 记忆层和文字去 AI 化，都在过去 7 天内快速积累了大量星标。以下是本期精选五强。

## 一、Russell-cell/PPT-Design-Prompt

⭐ 830 ｜ Python ｜ Russell-cell

专为 AI 幻灯片生成设计的提示词合集，系统梳理了从布局排版到色彩搭配的 PPT 设计提示词模板，可与主流 AI 编辑器配合使用。项目内容覆盖商务、学术、科技多种场景，帮助开发者和设计师快速生成高质量演示文稿。对于常需制作 PPT 又苦于提示词不精准的用户来说，是一份现成的「提示词速查手册」。

[项目地址](https://github.com/Russell-cell/PPT-Design-Prompt)

## 二、wuyoscar/gpt_image_2_skill

⭐ 817 ｜ Python ｜ wuyoscar

针对 OpenAI GPT Image 2 接口打造的提示词画廊与 CLI 工具集，内置大量经过实战验证的图片提示词模板，并支持作为 Agentic Skill 接入 Claude Code / Codex 等编辑器。用户可通过命令行直接调用图片生成能力，也可浏览提示词库挑选适合自己场景的模板。对比零散的社区分享，这是一份结构化、可复用的图像提示词资产库。

[项目地址](https://github.com/wuyoscar/gpt_image_2_skill)

## 三、epoko77-ai/im-not-ai

⭐ 514 ｜ Python ｜ epoko77-ai

专门用来给 AI 生成文字「去 AI 味」的改写技能。项目将 AI 润色文本，使其读起来更像人类写作，避免被检测工具识别。原版面向韩语场景，但技术方案具备通用性。在 AI 生成内容泛滥的时代，这一工具指向了内容创作领域一个真实的市场需求——如何让机器输出更接近人类表达的自然语感。

[项目地址](https://github.com/epoko77-ai/im-not-ai)

## 四、alash3al/stash

⭐ 422 ｜ Go ｜ alash3al

为 AI Agent 提供持久化记忆层的轻量 Go 库，支持存储剧情片段（Episodes）、事实（Facts）和工作上下文（Working Context）三种类型。解决了当前大多数 Agent 框架「会话结束即失忆」的痛点，让 Agent 能够跨会话积累知识、追踪状态。架构简洁、部署便捷，适合需要为自建 Agent 添加记忆能力的开发者快速集成。

[项目地址](https://github.com/alash3al/stash)

## 五、levelsio/superlevels

⭐ 420 ｜ JavaScript ｜ levelsio

独立开发者 @levelsio 亲手写的「超级 Chrome 扩展」，定位为比安装第三方陌生扩展更安全的个人增强工具集合。代码完全开源可审计，用户可以看到扩展所做的每一件事。项目体现了 levelsio 一贯的「自己用、公开分享」风格，也折射出近年开发者对主流扩展商店中闭源扩展安全性的普遍顾虑。

[项目地址](https://github.com/levelsio/superlevels)
