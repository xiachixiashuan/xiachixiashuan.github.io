# GitHub 周榜 Top5｜2026-05-02

本周 GitHub 新星爆发，Claude 生态、AI 交易、安全工具全面开花。以下 5 个项目均在过去 7 天内诞生或强势爆红，值得重点关注。

## 一、mattpocock/skills

⭐ 33.6k｜Shell｜作者：Matt Pocock（TypeScript 社区知名讲师）

专为工程师打造的 Claude 技能包集合，直接源自作者本人的 `.claude` 目录。涵盖代码审查、文档生成、重构助手等实用工作流，即插即用，无需复杂配置。**是目前 GitHub 上增速最快的 Claude 周边工具之一。**

[项目地址](https://github.com/mattpocock/skills)

## 二、Alishahryar1/free-claude-code

⭐ 12.9k｜Python｜作者：Alishahryar1

允许用户在终端、VS Code 扩展及 Discord 中免费使用 Claude Code，甚至支持语音输入。通过代理层绕过官方订阅限制，功能对标 OpenClaw。**在开发者社区引发热议，短期内积累近万三星。**

[项目地址](https://github.com/Alishahryar1/free-claude-code)

## 三、Z4nzu/hackingtool

⭐ 7.9k｜Python｜作者：Z4nzu

集合式渗透测试工具箱，一键安装 70+ 主流安全工具，覆盖信息收集、漏洞扫描、密码破解、无线攻击等全流程。**菜单驱动界面设计，大幅降低安全学习门槛。**

[项目地址](https://github.com/Z4nzu/hackingtool)

## 四、TauricResearch/TradingAgents

⭐ 6.2k｜Python｜作者：TauricResearch

基于多智能体 LLM 协作的量化交易框架，多个 AI Agent 分工负责行情分析、风险控制与策略执行。支持接入主流大模型，框架设计灵活可扩展。**是 AI+金融 赛道本周最受关注的开源项目。**

[项目地址](https://github.com/TauricResearch/TradingAgents)

## 五、abhigyanpatwari/GitNexus

⭐ 5.4k｜TypeScript｜作者：abhigyanpatwari

零服务器的代码智能引擎，在客户端构建 GitHub 仓库知识图谱，实现代码语义搜索与依赖追踪。**无需部署后端，纯前端即可完成复杂代码库分析。**

[项目地址](https://github.com/abhigyanpatwari/GitNexus)
