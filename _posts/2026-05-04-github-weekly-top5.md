# GitHub 周榜 Top5｜2026-05-04

本周 GitHub 涌现出一批值得关注的新项目——从航空安全数据开放到本地大模型部署社区，从苹果模拟器开发工具到 AI 辅助翻译技能，技术社区的创造力持续爆发。以下是过去 7 天内新创建、获星数最多的 5 个项目速览。

## 一、wrongly-cuddly-obsession/NTSB_FOIA_MU5735

⭐808｜文档数据｜wrongly-cuddly-obsession

MU5735 是 2022 年 3 月中国东方航空发生的空难事故，共造成 132 人罹难。这个仓库汇总了通过美国《信息自由法》（FOIA）向美国国家运输安全委员会（NTSB）申请获得的全部调查文件，为研究人员、航空安全专家及公众提供可检索的原始档案。在信息透明化日益受到重视的今天，这类数据开放项目具有重要的公众意义。

[项目地址](https://github.com/wrongly-cuddly-obsession/NTSB_FOIA_MU5735)

## 二、EvanBacon/serve-sim

⭐553｜TypeScript｜EvanBacon（Expo 核心开发者）

`serve-sim` 定位为"Apple 模拟器的 `npx serve`"——只需一行命令，即可快速启动 iOS / iPadOS 模拟器并提供应用服务，彻底省去繁琐的 Xcode 配置。对于日常进行 React Native 或 Expo 开发的工程师而言，这个工具大幅压缩了调试启动时间，让本地开发体验更接近 Web 开发的即时反馈感。

[项目地址](https://github.com/EvanBacon/serve-sim)

## 三、vibeforge1111/keep-codex-fast

⭐514｜Python｜vibeforge1111

这是一个为 Codex CLI 量身打造的技能（Skill），核心理念是"备份优先"：在保持本地 Codex 状态快速响应的同时，自动清理冗余缓存、定期备份工作状态，防止因本地数据损坏或膨胀导致 Codex 变慢甚至崩溃。对于深度依赖 Codex 做日常编程的开发者，这个技能可以无缝集成到现有工作流中。

[项目地址](https://github.com/vibeforge1111/keep-codex-fast)

## 四、noonghunna/club-3090

⭐445｜Shell｜noonghunna

消费级 GPU 部署 LLM 的社区实战手册。`club-3090` 汇集了社区贡献者在 RTX 3090 上运行大语言模型的经过验证的配置配方，支持 vLLM、llama.cpp、SGLang 等多个推理引擎，并提供开箱即用的 Shell 脚本。手头有闲置 3090 却不知从何入手跑模型的朋友，这个仓库是最实用的起点。

[项目地址](https://github.com/noonghunna/club-3090)

## 五、Cuimao777/cuimao-translator

⭐332｜Claude Code Skill｜Cuimao777

一个为中文读者打磨的 Claude Code 技能：一键将英文 PDF 翻译为流畅地道的中文。区别于简单的机器翻译，该技能借助 Claude 的语言理解能力对技术文档、学术论文等进行语义级别的优化，输出符合中文阅读习惯的译文，无需额外配置即可直接调用。

[项目地址](https://github.com/Cuimao777/cuimao-translator)
