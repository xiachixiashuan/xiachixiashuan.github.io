# GitHub 周榜 Top5｜2026-04-24

本周 GitHub 涌现大量高星新项目，从开源跨平台文档编辑器到 AI Agent 持久记忆层，从网页设计增强技能到游戏 Agentic 框架，创新层出不穷。以下是本周最值得关注的 5 个新开源项目。

## 一、golbin/hop

⭐ 920 ｜ TypeScript ｜ golbin

HOP 是一款专为 HWP/HWPX 格式文档（韩国主流办公文档标准）打造的开源跨平台桌面应用，支持 macOS、Windows 和 Linux。项目基于 rhwp 引擎构建，提供文件打开/保存、PDF 导出、打印对话框、拖拽打开、文件类型关联等完整 OS 集成功能。对于需要处理韩国政府、企业文档的开发者或用户，这是目前最完善的开源解决方案之一，MIT 许可证授权。

[项目地址](https://github.com/golbin/hop)

## 二、masterking32/MasterHttpRelayVPN

⭐ 903 ｜ Python ｜ masterking32

基于 Google Apps Script 实现域前置（Domain Fronting）的 HTTP/SOCKS5 代理工具，集成 MITM TLS 拦截、HTTP/1-2 多路复用与 DPI 绕过技术。通过将流量中继至 Google 基础设施，有效规避深度包检测，适合网络安全研究与隐私保护场景，纯 Python 实现、部署简便。

[项目地址](https://github.com/masterking32/MasterHttpRelayVPN)

## 三、ConardLi/web-design-skill

⭐ 739 ｜ CSS ｜ ConardLi

一个 AI Agent 设计技能插件，灵感来自 Claude Design，旨在将 AI 生成的网页从"功能可用"提升至"视觉精良"。通过向 AI 助手注入专业设计规则与审美标准，让编码助手在生成前端代码时自动遵循现代设计原则，无需人工反复调整样式细节，兼容主流 AI 编程工具。

[项目地址](https://github.com/ConardLi/web-design-skill)

## 四、leigest519/OpenGame

⭐ 729 ｜ TypeScript ｜ leigest519

OpenGame 是一个面向游戏开发的开放式 Agentic 编程框架，将大模型代理与游戏逻辑深度结合。基于 TypeScript 实现，支持 AI Agent 在游戏环境中自主感知状态、决策行动，为游戏 AI 研究、自动化测试与智能 NPC 设计提供了开放性基础架构。

[项目地址](https://github.com/leigest519/OpenGame)

## 五、Einsia/OpenChronicle

⭐ 640 ｜ Python ｜ Einsia

OpenChronicle 是 OpenAI Chronicle 的开源替代方案，为支持工具调用的 LLM Agent 提供本地优先的持久化记忆层。项目运行在本地 Mac 上，自动捕获屏幕与应用上下文，生成结构化 Markdown 记忆文件，记录正在处理的工作、已做的决策、常用工具与相关项目。兼容 Ollama、LM Studio、OpenAI、Anthropic 等多种模型提供商，MCP 客户端可直接接入，MIT 许可证开源。

[项目地址](https://github.com/Einsia/OpenChronicle)
